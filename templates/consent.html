My GitHub {{ project_year }} is a service that analyzes and visualizes your GitHub activity statistics for a specified year. By using this service, you consent to the following:

Data Collection and Processing

  • Data Processing: We will analyze your GitHub activity to generate statistics including but not limited to: commit patterns, repository contributions, programming language usage, and activity trends.
